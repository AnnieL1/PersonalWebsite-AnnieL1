The following projects have been created throughout the She Codes bootcamp:

This website!

My personal website was the first project we completed as part of She Codes and it gave me a sense of how to code. The website has been created in HTML and CSS.

Weather Project

The 'weather project' was created in Python and involved completing 8x problems which were ran against test scripts. It was extremely satisfying to receive an 'OK' from all test scripts.

She Codes News Room

News Room was written entirely in Django. Creating this website taught me the basics of Django including how to use inbuilt Django functionalities like @property.

gofundMEEPLE

gofundMEEPLE connects boardgame creators and enthusiasts together to bring boardgames to life. Backend is written with Django REST Framework. Frontend written in React. This website is the first fullstack web development project I've worked on and has taught me the importance of building the backend correctly so that data can be returned nicely in the frontend.

Learnable

Learnable is a skill-sharing platform aimed at making it easier for mentors to volunteer their time to teach others how to code. This website has been created as part of the group project and has highlighted the importance of git when working with others.
